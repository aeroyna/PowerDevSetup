# Setup OhMyZsh

## 1. Install Zsh

Ubuntu:

```bash
sudo apt update
sudo apt install zsh
chsh -s $(which zsh)
```

macOS:

```bash
brew install zsh
chsh -s /usr/local/bin/zsh   # or $(which zsh) if Homebrew path differs
```

Restart the terminal.

---

## 2. Install Oh-My-Zsh

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

Restart the terminal.

---

## 3. Recommended system tools (optional but useful)

### fzf

Ubuntu:

```bash
sudo apt install fzf
```

macOS:

```bash
brew install fzf
$(brew --prefix)/opt/fzf/install
```

### docker (optional)

Ubuntu:

```bash
sudo apt install docker.io
sudo usermod -aG docker "$USER"
```

macOS: install Docker Desktop.

### kubectl (optional)

Ubuntu:

```bash
sudo apt install kubectl
```

macOS:

```bash
brew install kubectl
```

### pyenv build dependencies (Ubuntu)

```bash
sudo apt update
sudo apt install -y build-essential \
  libssl-dev zlib1g-dev libbz2-dev libreadline-dev libsqlite3-dev \
  libncursesw5-dev xz-utils tk-dev libxml2-dev libxmlsec1-dev \
  libffi-dev liblzma-dev
```

---

## 4. Install pyenv

Ubuntu / Linux:

```bash
curl https://pyenv.run | bash
```

macOS:

```bash
brew install pyenv
```

---

## 5. Configure pyenv for login shells

Edit `~/.profile`:

```bash
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init --path)"
```

Edit `~/.zprofile`:

```bash
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init --path)"
```

Restart your login session or reboot.

---

## 6. Extra pyenv init for interactive zsh

Edit `~/.zshrc` and add (near the bottom):

```bash
if command -v pyenv 1>/dev/null 2>&1; then
  eval "$(pyenv init -)"
fi
```

---

## 7. Install Oh-My-Zsh plugins

Clone autosuggestions:

```bash
git clone https://github.com/zsh-users/zsh-autosuggestions \
  ~/.oh-my-zsh/custom/plugins/zsh-autosuggestions
```

Clone syntax highlighting:

```bash
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git \
  ~/.oh-my-zsh/custom/plugins/zsh-syntax-highlighting
```

Other plugins like z, extract, git, colored-man-pages, history-substring-search, docker, kubectl, pip, pyenv, and fzf are already available via Oh-My-Zsh.

---

## 8. Enable plugins in ~/.zshrc

Edit `~/.zshrc`:

```bash
plugins=(
  git
  z
  extract
  pip
  pyenv
  docker
  kubectl
  fzf
  colored-man-pages
  history-substring-search
  zsh-autosuggestions
  zsh-syntax-highlighting
)
```

Note:

`zsh-syntax-highlighting` should be last.

`zsh-autosuggestions` should come after history-substring-search.

Restart the terminal or run:

```bash
source ~/.zshrc
```

---

## 9. Install Python versions with pyenv

Example:

```bash
pyenv install 3.12.1
pyenv install 3.11.6
```

Set global default:

```bash
pyenv global 3.12.1
exec $SHELL -l
python --version
```

Set per-project (inside the project directory):

```bash
pyenv local 3.11.6
```

---

## 10. Basic checks

Autosuggestions: type a known command, suggestion should appear.

Syntax highlighting: command lines should be colored.

fzf: press Ctrl-R to search history.

z: after some directory changes, run `z <fragment>`.

extract: run `extract file.tar.gz`.

colored-man-pages: `man ls` should be colored.
